# gl-financas
📊  GL Finanças - Dashboard Financeiro

O **GL Finanças** é uma aplicação web intuitiva e responsiva para o controle de finanças pessoais. Com ele, você pode registrar entradas e saídas, visualizar o saldo total e gerenciar suas transações de forma rápida e eficiente.

---

## 📋 Funcionalidades

* **Resumo Geral:** Dashboard com visualização de Entradas, Saídas e Saldo Total.
* **Gestão de Transações:** Adição de novas movimentações com descrição, valor e tipo.
* **Feedback Visual:** Diferenciação de cores para valores positivos (verde) e negativos (vermelho).
* **Gestão em Tempo Real:** Possibilidade de excluir transações com atualização automática dos cálculos.
* **Design Responsivo:** Interface adaptada para Desktop e dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as tecnologias fundamentais da Web (Vanilla Stack):

* **HTML5:** Estrutura semântica e formulários.
* **CSS3:** Estilização, Flexbox, efeitos de hover e Media Queries para responsividade.
* **JavaScript:** Lógica de negócios, manipulação dinâmica do DOM e cálculos de saldo.
