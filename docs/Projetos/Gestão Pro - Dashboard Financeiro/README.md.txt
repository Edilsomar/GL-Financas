🚀 GL Finanças - Dashboard Financeiro
O GL Finanças é uma aplicação web simples e intuitiva para o controle de finanças pessoais. Com ele, você pode registrar entradas e saídas, visualizar o saldo total e gerenciar suas transações de forma rápida e eficiente.

📋 Funcionalidades
Resumo Geral: Visualização dinâmica de Entradas, Saídas e Saldo Total.

Gestão de Transações: Adição de novas transações com descrição, valor e tipo (Entrada/Saída).

Histórico: Tabela detalhada com todas as movimentações registradas.

Feedback Visual: Cores indicativas para valores positivos (verde) e negativos (vermelho).

Persistência de Dados (Interface): Remoção de itens da lista em tempo real com atualização automática dos cálculos.

Design Responsivo: Adaptado para dispositivos móveis e desktop.

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web:

HTML5: Estruturação semântica da página.

CSS3: Estilização, layout (Flexbox/Grid) e responsividade via Media Queries.

JavaScript (Vanilla): Lógica de negócios, manipulação do DOM e cálculos matemáticos.

📸 Demonstração
[!TIP]
Como usar:

Clique em "+ Nova Transação".

Digite a descrição (ex: "Salário" ou "Aluguel").

Insira o valor.

Selecione se é uma Entrada ou Saída e clique em "Enviar".

🚀 Como Executar o Projeto
Faça o clone do repositório:

Bash
git clone https://github.com/edilsomar/gl-financas.git
Navegue até a pasta do projeto.

Abra o arquivo index.html em qualquer navegador moderno.

📁 Estrutura de Arquivos
index.html: Arquivo principal da interface.

style.css: Estilização principal e layout desktop.

cssandroid.css: Ajustes específicos para dispositivos móveis.

javascript.js: Toda a inteligência e manipulação da página.

img/ e icones/: Recursos visuais do projeto.
